aspose包，处理文档

com.aspose.words.Document (word)
com.aspose.words.PdfSaveOptions; (word保留为PDF时候需要引用的包)

在word中插入自定义属性
1。点击 文件→属性→高级属性→自定义 可以定义属性和属性节点
2.在word中插入自定义文档 鼠标移动到光标位置 Ctrl+F9 ，这时候会在光标位置插入一个 {  }所圈起来的表达式，在其中输入
{ DOCPROPERTY 属性名 }，即可吧属性插入其中，“点击表达式按F9，可以将表达式刷新为定义属性时候设置的值”

二，读取并且设置自定义属性，并且刷新word中自定义属性的方法
//读文档
Document doc = new Document(文档路径)；
//获取文档所有属性
com.aspose.words.CustomDocumentProperties propertise = doc.getCustomDocumentProperties();

//设定自定义属性
properties.get("属性名称，汉族有unicode码替代").setValue("新输入值");

//将word文档保存为PDF的设置
PdfSaveOptions options = new PdfSaveOptions();
options.setFontEmbeddingMode(1); //参数说明0,为默认模式，1为兼容模式，推荐1，因为不同操作系统上由于字体不全的原因，直接转pdf可能会导致文档中汉字缺失

//
doc.updateFields();//更新文档，刷新域
doc.save(outputPath,options);//保存 （说明：outputPath需要见输出路径写完全，包括文档名称和文档的后缀，转存为pdf的话，将路径文档后缀转换成pdf就行，options是前面定义的PDF设置）



关于字体的说明:windows系统一般汉字字体是全的，linux字体可能不全，可通过一下语句查询
